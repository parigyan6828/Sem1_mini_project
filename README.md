## 🚀 About The Project

💡 An interactive **Typing Speed Test** built using **C++ and Object-Oriented Programming** principles.
It challenges users to type predefined text while measuring **speed (WPM)** and **accuracy**, and ranks them on a leaderboard.

---

## ✨ Features

🔥 Real-time typing test
⚡ Accurate WPM calculation
🎯 Precision-based accuracy tracking
🏆 Leaderboard ranking system
🔁 Replay option for continuous practice

---

## 🧠 OOP Design

🔹 **Player Class** → Handles user data & performance
🔹 **Test Class** → Manages typing logic & timing
🔹 **Leaderboard Class** → Sorts & displays rankings
